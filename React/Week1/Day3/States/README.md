# States 
- React components has a built-in state object
 - The state object is where you store property values that belongs to the component.
 - When the state object changes, the component re-renders.


## States are Variables that holds data 
- React Renders the toptobottom ui only once , so it doesnt re render the variable you want to change dynamically 
- how to declare states `const [name , changeName]= useState(initialvaluehere)`
