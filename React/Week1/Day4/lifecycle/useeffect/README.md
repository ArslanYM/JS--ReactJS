# Component LifeCycle and Use Effect Hook

- Lifecycle explains how everthing works in a react app component from birth till death
- There are three stages : 
 - Mounting 
 - Updating 
 - Unmounting

## Use Effect 
- Controls ui  depending upon which stage of lifecycle youre in 
-  The useEffect Hook allows you to perform side effects in your components. 
- useEffect accepts two arguments. The second argument is optional.
 - `useEffect(<function>, <dependency>)`
